# 📊 SQL - Análise de Vendas

Projeto de **Análise de Dados utilizando SQL**, simulando cenários reais de negócio para extração de insights a partir de um banco de dados de vendas.

O objetivo deste projeto é demonstrar habilidades em **modelagem de dados, consultas SQL e análise de métricas de negócio**, utilizando consultas semelhantes às usadas por **Analistas de Dados em empresas**.

# 🗄️ Estrutura do Banco de Dados

O banco de dados simula um sistema de **vendas de uma empresa**.

### Tabelas principais

* **clientes**
* **produtos**
* **vendedores**
* **vendas**
* **datas**

Cada tabela representa uma parte do processo de vendas.

---

# 📊 Análises realizadas

O projeto contém **50 consultas SQL**, incluindo análises como:

### 📦 Produtos

* Produtos mais vendidos
* Produtos mais lucrativos
* Ranking de produtos

### 👥 Clientes

* Clientes que mais compram
* Clientes por faixa etária
* Clientes por cidade e estado

### 💰 Vendas

* Faturamento total
* Ticket médio
* Quantidade total vendida

### 🧑‍💼 Vendedores

* Ranking de vendedores
* Vendedores com maior faturamento
* Performance de vendas

### 📅 Análises de tempo

* Faturamento mensal
* Vendas por período
* Tendência de vendas
